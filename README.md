# Tetris-Qt #
Classic tetris game made with c++ and qt5.11

![alarum](https://github.com/walidAbbassi/Tetris-Qt/blob/master/picture/tetris.png)

# Control 
```
key → :  move left 
```
```
key ← :  move right 
```
```
key ↓ : rotate right 
```
```
key ↑ :  rotate left 
```
```
key space :  drop down 
```
```
key D :  drop one line down
```
